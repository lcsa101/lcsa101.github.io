# 二分查找

二分查找是在有序数组中查找某一特定元素的搜索算法

## 1.使用前提
 - 必须是数组
 - 必须有序

## 2.查找过程

 假设有一有序数组，且按照从小到大排列，例如

 ```
 -------------------------------------
| 1 | 3 | 10 | 44 | 53 | 68 | 75 | 94 |
 -------------------------------------
  0   1    2    3   4     5    6    7  <—— 数组下标
 ```

定义两个下标指针，分别指向要查找范围的左端与右端，定义mid为(left+right)/2，若为小数，则向下取整

 ```
 -------------------------------------
| 1 | 3 | 10 | 44 | 53 | 68 | 75 | 94 |
 -------------------------------------
  0   1    2    3   4     5    6    7    
 left          mid                 right
 ```

现在，查找元素10

mid对应元素为44，而44大于10，所以right指针左移至mid-1处

```
 -------------------------------------
| 1 | 3 | 10 | 44 | 53 | 68 | 75 | 94 |
 -------------------------------------
  0   1    2    3   4     5    6    7    
 left mid right
 ```

 此时，新的mid在数组下标1处，对应元素为3，3小于10，则left指针移至mid+1处

 ```
 -------------------------------------
| 1 | 3 | 10 | 44 | 53 | 68 | 75 | 94 |
 -------------------------------------
  0   1    2    3   4     5    6    7    
         right
         left
         mid
 ```

此时，right，left，与mid重合，且所指元素为要查找的元素，执行完毕


**程序实例**


给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 O(log n) 的算法。

 

示例 1:

输入: 
```
nums = [1,3,5,6], target = 5
```
输出: 
```
2
```
示例 2:

输入:
```
 nums = [1,3,5,6], target = 2
```
输出:
```
 1
```
示例 3:

输入: 
```
nums = [1,3,5,6], target = 7
```
输出:
```
 4
```

提示:
```
1 <= nums.length <= 104
-104 <= nums[i] <= 104
nums 为 无重复元素 的 升序 排列数组
-104 <= target <= 104
```

```cpp
#include<iostream>
#include<vector>
using namespace std;
int main()
{
	int n, tar;
	cin >> n;
	vector<int> ans(n);
	for (int i = 0; i < n; i++)
		cin >> ans[i];
	cin >> tar;
	int left = 0;
	int right = n;
	while (left < right)
	{
		int mid = (left + right) / 2;
		if (ans[mid] < tar)
			left = mid + 1;
		else
			right = mid;
	}
	cout << left;
}
```

## 3.二分查找例题

**P2249 【深基13.例1】查找**

题目描述

输入 n 个不超过 10^9 的单调不减的（就是后面的数字不小于前面的数字）非负整数a<sub>1</sub>,a<sub>2</sub>...a<sub>n</sub>，然后进行 m 次询问。对于每次询问，给出一个整数 q，要求输出这个数字在序列中第一次出现的编号，如果没有找到的话输出 -1 。

输入格式

第一行 2 个整数 n 和 m，表示数字个数和询问次数。

第二行 n 个整数，表示这些待查询的数字。

第三行 m 个整数，表示询问这些数字的编号，从 1 开始编号。

输出格式

输出一行，m 个整数，以空格隔开，表示答案。

输入输出样例 #1

输入 #1

```
11 3
1 3 3 3 5 7 9 11 13 15 15
1 3 6
```

输出 #1

```
1 2 -1 
```

*说明/提示*

数据保证，1 ≤ n ≤ 10^6，0 ≤ a<sub>i</sub>,q ≤ 10^9，1 ≤ m ≤ 10^5

本题输入输出量较大，请使用较快的 IO 方式。

```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;
int findnum(vector<int>& a, int b)
{
	int left = 0;
	int right = a.size() - 1;
	int ans = -1;
	while (left <= right)
	{
		int mid = (left + right) / 2;
		if (a[mid] >= b)
		{
			ans = mid;
			right = mid - 1;
		}
		else
		{
			left = mid + 1;
		}
	}
	if (ans != -1 && a[ans] == b)
		return ans;
	else
		return -1;
}
int main()
{
	int m, n;
	cin >> n >> m;
	vector<int> num(n);
	for (int i = 0; i < num.size(); i++)
		cin >> num[i];
	vector<int> tarnum(m);
	for (int i = 0; i < tarnum.size(); i++)
		cin >> tarnum[i];
	for (int i = 0; i < tarnum.size(); i++)
	{
		int pos = findnum(num, tarnum[i]);
		if (i)
			cout << " ";
		if (pos == -1)
			cout << -1;
		else
			cout << pos + 1;
	}
	return 0;
}
```

## 关于二分查找的几种写法

以下列举二分查找两种比较规范的写法，但不排除其他简便写法

以下以找指定数字为例，若找到则返回数组下标，否则返回-1

**1.区间左闭右闭**

区间左闭右闭即形如：[1,2]

```cpp
...
int l = 0;
int r = vec.size() - 1;//因为区间为左闭右闭，故数组最后一元素也是合法的，可以取到，所以右指针指向数组最后一位下标
while(l <= r)//区间左闭右闭，区间端点可以相等，类似[1,1]
{
	int mid = (l + r)/ 2
	if(vec[mid] > tar)
	{
		r = mid - 1;//此时，mid所指的元素大于目标值，左移右指针至mid-1处（因为区间左闭右闭，mid处已经被包含，若取r = mid则区间被重复考虑）
	}
	else if(vec[mid] < tar)
	{
		l = mid + 1;
	}
	else
	{
		return mid;
	}
}
return -1;
...
```

**2.区间左闭右开**

区间左闭右开即形如：[1,2)

```cpp
...
int l = 0;
int r = vec.size();//因为区间为左闭右开，故数组最后一元素不合法
while(l < r)//区间左闭右开，区间端点不可以相等，类似[1,1)
{
	int mid = (l + r)/ 2
	if(vec[mid] > tar)
	{
		r = mid;//此时，mid所指的元素大于目标值，左移右指针至mid处（因为区间左闭右开，mid处未被包含，则取r = mid合法）
	}
	else if(vec[mid] < tar)
	{
		l = mid + 1;
	}
	else
	{
		return mid;
	}
}
return -1;
...
```

# 二分答案

## 1.二分答案简要步骤
1）依据题目估计最终答案范围是什么
2）分析问题的答案与给定条件之间的单调性
3）建立函数，在答案固定的条件下，判断给定前提是否达标
4）在可能的答案范围内，不断二分查找，每次用函数判断，直到二分结束

## 2.与二分答案有关例题

**P1873 [COCI 2011/2012 #5] EKO / 砍树**

题目描述

伐木工人 Mirko 需要砍 M 米长的木材。对 Mirko 来说这是很简单的工作，因为他有一个漂亮的新伐木机，可以如野火一般砍伐森林。不过，Mirko 只被允许砍伐一排树。

Mirko 的伐木机工作流程如下：Mirko 设置一个高度参数 H（米），伐木机升起一个巨大的锯片到高度 H，并锯掉所有树比 H 高的部分（当然，树木不高于 H 米的部分保持不变）。Mirko 就得到树木被锯下的部分。例如，如果一排树的高度分别为 20,15,10 和 17，Mirko 把锯片升到 15 米的高度，切割后树木剩下的高度将是 15,15,10 和 15，而 Mirko 将从第 1 棵树得到 5 米，从第 4 棵树得到 2 米，共得到 7 米木材。

Mirko 非常关注生态保护，所以他不会砍掉过多的木材。这也是他尽可能高地设定伐木机锯片的原因。请帮助 Mirko 找到伐木机锯片的最大的整数高度 H，使得他能得到的木材至少为 M 米。换句话说，如果再升高 1 米，他将得不到 M 米木材。

输入格式

第 1 行 2 个整数 N 和 M，N 表示树木的数量，M 表示需要的木材总长度。

第 2 行 N 个整数表示每棵树的高度。

输出格式

1 个整数，表示锯片的最高高度。

输入输出样例 #1

输入 #1

```
4 7
20 15 10 17
```

输出 #1

```
15
```

输入输出样例 #2

输入 #2

```
5 20
4 42 40 26 46
```

输出 #2

```
36
```

*说明/提示*

对于 100% 的测试数据，1≤N≤10^6 ，1≤M≤2×10^9 ，树的高度 ≤4×10^4 ，所有树的高度总和 >M。


```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;
//编写函数，计算被砍木材长度总和
long long func(vector<int>& a, int h)//h为刀片高度
{
	long long total = 0;//用long long避免溢出
	for (int i = 0; i < a.size(); i++)
	{
		if (a[i] > h)
			total += (a[i] - h);
	}
	return total;
}
int main()
{
	int m, n;
	cin >> n >> m;
	vector<int> treeheight(n);
	int l = 1;
	int r = 0;
	for (int i = 0; i < n; i++)
		cin >> treeheight[i];
	//确定答案范围，为[1,max]
	for (int i = 0; i < n; i++)
		r = max(treeheight[i], r);
	//使数组有序，即满足单调性，以便后续二分
	sort(treeheight.begin(), treeheight.end());
	int h = 1;
	//不断二分查找
	while (l <= r)
	{
		int mid = (l + r) / 2;
		long long tlong = func(treeheight, mid);//最开始用mid高度切
		if (tlong >= m)
		{
			h = mid;
			l = mid + 1;
		}
		else
		{
			r = mid - 1;
		}
	}
	cout << h << endl;
	return 0;
}
```

# 三分查找

## 1.三分查找简介

三分法原理与二分法类似，二分法将区间一分为二，而三分法将区间分为三份进行查找

从数学角度来看，二分法适用于单调函数，即一次函数，数组元素满足单调性，即单调递增或单调递减
而三分法常适用于单峰函数寻找极值，即二次函数的极值

![](images/2026-07-16-11-00-01.png)

如图，三分法用两个端点（记为m<sub>1</sub>和m<sub>2</sub>）将区间分为三份，要求极值，用极限的思想逼近
记上面的函数为y = f(x)，由图像可知，f(m<sub>1</sub>) > f(m<sub>2</sub>)，若要趋近于极值，则令l = m<sub>1</sub>，即缩小区间至[m<sub>1</sub>,r]

但是，如果m<sub>2</sub>在极值左边怎么办？即极值不在m<sub>1</sub>与m<sub>2</sub>所框定的范围内，如图

![](images/2026-07-16-15-07-03.png)

f(m<sub>1</sub>) > f(m<sub>2</sub>)，此时，参照第一种情况，抛去区间[l,m<sub>1</sub>]，不会影响极值点

那么，m<sub>1</sub>，有没有可能位于极值点左侧呢？答案是不可能的，因为若m<sub>1</sub>位于左侧，则f(m<sub>1</sub>) < f(m<sub>2</sub>)，假设不成立

**由此可见，三分法就是通过比较两个值的大小，不断缩小1/3区间，直至找到结果**

## 2.三分法过程

![](images/2026-07-16-15-20-22.png)

如图，图像被分为[left,midl]，[midl,midr]，[midr,right]

```cpp
midl = left + (right - left)/ 3;
midr = right - (right - left)/ 3;
```

理解：（right - left）/ 3即为整段区间取1/3大小，m<sub>1</sub>坐标即为左端点坐标加区间1/3长度，m<sub>2</sub>坐标即为右端点坐标减1/3区间长度
若midl更靠近极值点，则令right = midr - 1（抛去远离的那一段）
若midr更靠近极值点，则令left = midl - 1（抛去远离的那一段）

三分查找的时间复杂度为：O(log<sub>3</sub>n)

## 3.代码示例
寻找整数区间内函数极大值

```cpp
#include <iostream>
using namespace std;
// 示例函数
int calc(int x) {
   return -1 * (x - 3) * (x - 3) + 9; 
}
//搜索实现
int main() {
   int l = 0, r = 10; // 搜索区间
   while (l < r) {
       int lmid = l + (r - l) / 3;
       int rmid = r - (r - l) / 3;
       if (calc(lmid) < calc(rmid)) {
           l = lmid + 1;
       } else {
           r = rmid - 1;
       }
   }
   cout << "极大值为: " << calc(l) << endl;
   return 0;
}
```
